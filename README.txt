README

Make sure you have the required .csv files. The Jupiter notebook and the .csv files should be located in the same folder.
Make sure you install all the necessary packages before importing them.

1. Use Anaconda Python Distribution to run Jupyter Notebook 
	https://www.anaconda.com/distribution/


2. Install the required packages using the following syntax:
	 $ pip install ‘package_name’

	example: $ pip install uszipcode 
	
	documentation link: https://pypi.org/project/uszipcode/ 


3.  Required packages list : NumPy, Pandas, Matplotlib, Seaborn, uszipcode, warnings
